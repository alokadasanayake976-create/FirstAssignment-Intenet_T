Info-about Html Tags

1. html

Defines that the document is written using HTML markup language. It is the root element of an HTML page.

2. head

Contains metadata, page title, links to stylesheets, and other information not displayed on the webpage.

3. meta

Provides metadata such as character encoding and viewport settings for responsive design.

4. title

Specifies the title of the webpage shown in the browser tab.

5. link

Connects external resources such as CSS files to the HTML document.

6. body

Contains all the visible content displayed on the webpage.

7. nav

Defines a navigation section containing links to other pages or sections.

8. a

Creates hyperlinks that allow users to navigate to other pages, files, or websites.

9. h1

Defines the main heading of a webpage.

10. h2

Defines a second-level heading used for major sections.

11. h3

Defines a third-level heading used for subsections.

12. p

Defines a paragraph of text.

13. section

Groups related content into logical sections of a webpage.

14. img

Displays images on a webpage.

15. details

Creates a collapsible section that users can expand or hide.

16. summary

Defines the visible heading for a details element.

17. iframe

Embeds another webpage or external content such as YouTube videos.

18. table

Creates a table for organizing data into rows and columns.

19. tr

Defines a row inside a table.

20. td

Defines a data cell inside a table row.

21. center

Centers content horizontally on the page. (Deprecated in HTML5; CSS is recommended instead.)

22. embed

Embeds external content such as PDF documents into a webpage.

23. video

Embeds and plays video files directly within a webpage.

24. br

Inserts a line break and moves content to the next line.

25. b

Displays text in bold style without adding semantic importance.

26. div

Defines a division or section used for grouping content.

27. span

Used to style or group small portions of text.

28. ul

Creates an unordered (bulleted) list.

29. ol

Creates an ordered (numbered) list.

30. li

Defines a list item inside ul or ol.

31. form

Creates a form for collecting user input.

32. input

Creates an input field for user data.

33. label

Provides a label for a form control.

34. button

Creates a clickable button.

35. textarea

Creates a multi-line text input area.

36. select

Creates a drop-down list.

37. option

Defines an item in a drop-down list.

38. header

Represents introductory content of a page or section.

39. footer

Defines the footer section of a page or section.

40. main

Represents the main content of a webpage.
